## Project Context

Developer Swaleh Portfolio is a modern full-stack portfolio website built using:

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Supabase
- PostgreSQL

The application follows a minimal Vercel-inspired aesthetic with strong emphasis on:

- Clean architecture
- Accessibility
- Performance
- Scalability
- Responsive design
- Modern engineering standards

---

# Design Philosophy

Maintain:

- Minimalism
- Consistency
- Strong spacing systems
- Professional UI hierarchy
- Accessible interfaces
- Performance-first engineering

Avoid:

- Visual clutter
- Excessive animation
- Unnecessary dependencies
- Multiple accent colors
- Overengineered abstractions
