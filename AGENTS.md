# Animation Standards

Framer Motion should be used conservatively.

Allowed transitions:

- opacity
- translateY
- scale
- blur fade

Animations must:

- Improve UX
- Be subtle
- Respect performance
- Avoid distraction

Avoid:

- Bounce animations
- Excessive rotations
- Heavy parallax effects

---

# Accessibility Standards

Required:

- Semantic HTML
- Keyboard navigation
- Focus-visible states
- Proper labels
- Accessible contrast
- Reduced motion friendly interactions

---

# Performance Standards

Required:

- Lazy loading where applicable
- next/image optimization
- Minimal client-side JavaScript
- Reduced layout shift
- Responsive layouts
- Stable hydration
- No console warnings or errors

---

# SEO Standards

Every page should include:

- Metadata
- OpenGraph metadata
- Twitter metadata
- Semantic headings
- Descriptive titles
- Accessible page structure
